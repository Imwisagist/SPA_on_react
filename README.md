# Let's get started:

### Clone repository and enter in directory infra:
```bash
git clone https://github.com/Imwisagist/Test_task_for_AG.git && cd infra
```
### Run docker-compose:
```bash
docker-compose up -d
```
### Open your browser:
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
